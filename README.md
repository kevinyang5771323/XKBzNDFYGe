# 人事管理系统简介 java554

## 项目概述

本项目为人事管理系统，采用前后端分离的设计模式。前端使用 Vue.js 框架，后端基于 Spring Boot 构建而成，采用 MVC 架构，数据持久化通过 MyBatis 实现，数据库使用 MySQL。

## 技术栈

- **后端：**
  - Java
  - Spring Boot
  - MVC
  - MyBatis
  - MySQL
  - Maven
- **前端：**
  - HTML
  - JavaScript
  - CSS
  - Vue.js
- **开发工具：**
  - IntelliJ IDEA
  - VS Code

## 功能模块

人事管理系统包含但不限于以下功能模块：

- 员工信息管理
- 考勤管理
- 薪资管理
- 岗位管理
- 部门管理

## 系统角色

- 系统管理员
- 人事专员
- 员工

## 运行环境

- **后端：**
  - JDK 1.8 或更高版本
  - MySQL 5.7 或更高版本
- **前端：**
  - Node.js 14 或更高版本

## 部署步骤

1. 配置 IntelliJ IDEA 或 VS Code 开发环境。
2. 安装和配置 MySQL 数据库。
3. 安装 JDK 1.8 或更高版本。
4. 使用 Maven 构建后端项目。
5. 使用 Node.js 和相关工具集构建前端项目。
